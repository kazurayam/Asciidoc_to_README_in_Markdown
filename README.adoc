= Include directive is not supported in Github-flavored Asciidoc --- a workaround

In a GitHub repository, you can make a file named `README.adoc` in [Asciidoc](https://asciidoc-py.github.io/index.html) format. GitHub is able to convert Asciidoc text to HTML runtime so that it renders in browsers. You can enjoy Asciidoc capabilities to author your documents and publish them in GitHub. A sample is here: 

- 



However, there is a defect.

GitHub-flavored-Asciidoc does not support `include::file/path/to/include[]` directive.
